# Netflix Data Analysis

## Project Overview
This project analyzes Netflix content using Python and Pandas.

## Dataset
Netflix Titles Dataset

## Questions Explored

- Which country contributes the most content?
- How has Netflix content grown over time?
- Which genres are most common?
- Which ratings dominate Netflix?
- Which actors appear most frequently?

## Key Findings

### Country Analysis
- United States had the highest number of titles.
- India ranked second.

### Content Growth
- Content additions increased rapidly until 2019.
- A decline was observed during 2020–2021, possibly due to COVID-19 disruptions.

### Genre Analysis
- International Movies were the most common category.
- Dramas and Comedies were also highly represented.

### Rating Analysis
- TV-MA was the dominant rating category.

### Cast Analysis
- Anupam Kher appeared most frequently in the dataset.
